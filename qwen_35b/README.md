# Paris Tourism Website 🗼

A beautiful, responsive travel city advertisement website designed to showcase Paris as a premier tourist destination.

## Features

- **Modern & Responsive Design**: Looks great on all devices (desktop, tablet, mobile)
- **Smooth Animations**: Engaging scroll animations and transitions
- **Interactive Gallery**: Lightbox functionality for image viewing
- **Contact Form**: User-friendly form for travel inquiries
- **Mobile Navigation**: Hamburger menu for smaller screens
- **Parallax Effects**: Visual depth in hero section

## Pages/Sections

1. **Home (Hero)**: Eye-catching landing with call-to-action
2. **About**: Introduction to Paris as a destination
3. **Attractions**: Featured landmarks and points of interest
4. **Gallery**: Photo showcase of the city
5. **Events**: Upcoming festivals and celebrations
6. **Contact**: Inquiry form for travel planning

## Technologies Used

- HTML5: Semantic markup structure
- CSS3: Modern styling with Flexbox/Grid, animations
- JavaScript (ES6+): Interactive features and DOM manipulation
- Google Fonts: Playfair Display & Poppins typography

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process required - it's ready to use!

## Customization

### Colors

Edit the CSS variables in `styles.css`:

```css
:root {
  --primary-color: #e74c3c;
  --secondary-color: #2c3e50;
  --accent-color: #f39c12;
}
```

### Content

- Update text content in `index.html` for different cities or information
- Replace image URLs with your own photos
- Modify events and attractions as needed
